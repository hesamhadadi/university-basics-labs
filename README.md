# University Basics — Coursework Lab Exercises

A consolidated collection of early coursework exercises from my **B.Sc. in Computer Engineering at Khayyam University** (Mashhad, Iran) and other foundational labs.

These are small, self-contained lab assignments covering web fundamentals, OOP, and theory of computation. They are not production projects — they're kept here as a historical portfolio of learning.

## Structure

### `web-basics/`

| Folder | Topic | Stack |
|--------|-------|-------|
| `resume-html/` | Personal resume page — first HTML assignment | HTML |
| `trello-clone/` | Static Trello UI clone | HTML · CSS |
| `photo-album-vue/` | Photo album SPA | Vue 2 · JavaScript |

### `php-basics/`

jQuery + PHP + Bootstrap 4 mini-project with forms, file upload, image resizing, loops, and conditionals — early backend fundamentals.

### `typescript-basics/`

| Folder | Topic |
|--------|-------|
| `bank-account-oop/` | Getter/setter OOP exercise with Jest tests |
| `generic-validation/` | Generic validation system with rules composition |

### `theory-of-computation/`

| Folder | Topic |
|--------|-------|
| `turing-machine/` | Turing Machine simulator (tape + transition function) in Java |
